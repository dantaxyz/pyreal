Base Explainer¶

Explainer(algorithm, model, x_train_orig[, …])

Generic Explainer object

Explainer.fit()

Fit this explainer object.

Explainer.produce(x_orig)

Return the explanation, in the desired form.

Explainer.transform_to_x_explain(x_orig)

Transform x_orig to x_explain, using the e_transforms

Explainer.transform_to_x_model(x_orig)

Transform x_orig to x_model, using the m_transforms

Explainer.transform_to_x_interpret(x_orig)

Transform x_orig to x_interpret, using the i_transforms :param x_orig: Original input :type x_orig: DataFrame of shape (n_instances, x_orig_feature_count)

Explainer.model_predict(x_orig)

Predict on x_orig using the model and return the result

Explainer.feature_description(feature_name)

Returns the interpretable description associated with a feature

Explainer.convert_data_to_interpretable(x_orig)

Convert data in its original form to an interpretable form, with interpretable features :param x_orig: Input data to convert :type x_orig: DataFrame of shape (n_instances, n_features)
