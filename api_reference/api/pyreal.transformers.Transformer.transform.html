pyreal.transformers.Transformer.transform#

Transformer.transform(x)[source]#

Wrapper for data_transform. Transforms data x to a new feature space.

Included for compatibility with existing ML libraries. :param x: The dataset to transform :type x: DataFrame of shape (n_instances, n_features)

Returns:

The transformed dataset

Return type:

DataFrame of shape (n_instances, n_transformed_features)
